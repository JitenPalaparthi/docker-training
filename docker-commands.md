### To check whether docker is installed

```docker version```
```docker info```
### To pull an ubuntu image

```docker pull ubuntu```

### To check images

```docker image ls```
```docker images```

### To create a container
```docker run -i -t --name u1 ubuntu bash```

### To list running containers
```docker container ls```
```docker ps```

### To list all containers running and stopped
```docker ps -a```
